# Cyclistic Bike-Share Analysis

## Project Overview
This project is part of the Google Data Analytics Professional Certificate case study.

The goal of this analysis was to understand how annual members and casual riders use Cyclistic bikes differently.

## Tools Used
- Excel
- Tableau

## Key Insights
- Casual riders take longer rides
- Members ride more frequently
- Casual riders are more active during weekends

## Dashboard
The Tableau dashboard was created to visualize customer behavior patterns and business insights.
